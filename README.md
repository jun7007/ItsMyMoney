# ItsMyMoney

개인 주식 포트폴리오 관리 앱 — 매수/매도 이력, 실시간 손익, 종목 뉴스를 한곳에서 관리합니다.

## 주요 기능

- **한글 종목 검색 및 자동 등록** — 번거롭게 `005930.KS` 같은 코드 형식으로 외워 칠 필요 없이, 한글 회사명이나 티커로 편리하게 검색하고 선택하여 등록할 수 있는 자동완성 검색창을 제공합니다.
- **보유 종목 관리** — 한국(코스피/코스닥) + 미국 주식을 지원하며, 알파뉴메릭 티커(예: KODEX 미국에어로스페이스 `0167Z0.KS`)도 완벽히 지원합니다.
- **매수/매도 기록** — 거래 이력 저장 및 평균 단가 자동 계산.
- **안정적인 실시간 시세 연동** — 속도가 빠르고 에러율이 낮은 **네이버 금융 API**를 연동하여 실시간 시세를 가져오며, 대시보드와 보유 종목 화면에서 60초마다 자동 갱신됩니다.
- **현재가 갱신시간 표시** — 대시보드, 보유 종목 목록 카드 및 종목 상세 페이지에 시세가 마지막으로 업데이트된 시간(`갱신: 오전/오후 HH:MM`)을 직관적으로 보여줍니다.
- **친숙한 주식 시장 색상 적용** — 한국 주식 시장 통상 기준을 반영하여 **수익/매수는 빨간색(Red)**, **손실/매도는 파란색(Blue)**으로 일관되게 표현합니다.
- **종목 뉴스** — 종목별 관련 뉴스 조회.
- **SQLite 저장** — 모든 데이터 로컬 영구 보관.
- **모바일 접속** — 집 Wi-Fi 및 Tailscale로 외부 접속.

| 영역 | 기술 |
|------|------|

## 사용 방법

### 1. 종목 등록 및 자가 치유(Self-Healing)

**거래 기록** 탭에서 **종목명(한글)** 또는 **티커**를 입력하여 검색 후, 드롭다운에서 선택하여 바로 등록할 수 있습니다. 

* *기존에 회사명 대신 코드(`005930.KS`) 등으로 등록되어 있던 종목*은, 검색창에서 올바른 한글 이름(예: `삼성전자`)을 검색하여 선택하는 순간 백엔드 데이터베이스의 이름이 **올바른 한글 명칭으로 자동 정정 및 업데이트**되는 자가 치유 기능이 내장되어 있습니다.

| 시장 | 검색어 예시 | 티커 포맷 (자동 등록) |
|------|-----------|-----------|
| 미국 | `애플` 또는 `AAPL` | `AAPL` |
| 한국 코스피 | `삼성전자` 또는 `005930` | `005930.KS` |
| 한국 코스닥 | `카카오` 또는 `035720` | `035720.KQ` |
| 한국 ETF | `우주항공` 또는 `0167Z0` | `0167Z0.KS` |

### 2. 매수/매도 기록

종목 선택 → 매수/매도 선택 → 수량, 단가, 수수료, 거래일시 입력 → **거래 저장**

### 3. 손익 확인

**대시보드** 또는 **보유 종목** 탭에서 실시간 손익 및 현재가 마지막 갱신 시간 확인. 종목 카드를 누르면 상세 정보와 뉴스를 볼 수 있습니다.
